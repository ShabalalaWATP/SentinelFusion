# Agentic Upgrade Framework

This folder defines the project-level agent framework for improving Sentinel Fusion through orchestrated loops.

Use it when Alex asks for the Orchestrator Agent, an agentic upgrade loop, concurrent agents, or a broad product-improvement pass.

Start here:

- `AGENTIC_UPGRADE_FRAMEWORK.md`: operating model, loop, concurrency, gates, and decision rules.
- `ROLE_BRIEFS.md`: detailed duties and outputs for each agent.
- `SYSTEM_PROMPTS.md`: concise system/task prompts for the Orchestrator and subagents.
- `templates/agent-brief.md`: reusable brief for spawning a focused agent.
- `templates/goal-loop-state.md`: state record for ambitious, measurable improvement loops.

The framework is deliberately documentation-first. It should guide Codex, local subagents, or future external automation without hiding project rules in chat history.
